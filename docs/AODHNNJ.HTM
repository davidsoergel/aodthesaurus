NKdocument or presentation by content and format   d-out   qh
HN Introduced 1995.
SN Throughout this section, "document" is used broadly to include any kind of written or oral document or presentation. These descriptors are used for documents that belong to a given document type and for documents that discuss a given document type. For example, the descriptor *NM8 policy statement* qh ah would be used to index a policy statement as well as to index a guide on writing a policy statement.
      These descriptors can be used in one of two ways: (1) to index documents that discuss a format, such as a document that discusses the creation and structure of a clinical guide; or (2) to index the format of a document, such as a document that is a clinical guide. Some systems use a separate data field for this.
